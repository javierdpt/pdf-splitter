## Release Notes

### v1.0.0 (January 28, 2026)

- Initial release
- Cross-platform desktop application for splitting PDF files
- Easy-to-use graphical interface
- Customizable pages per document
- Standalone executables for macOS and Windows

## Installation Instructions

### macOS

1. Download the `PDF-Splitter-macos-v1.0.0.zip` file
2. Double-click the downloaded ZIP to extract `PDF-Splitter.app`
3. Move the `.app` into your `Applications` folder (or run from Finder)
4. If macOS blocks the app on first run, right-click the app and choose "Open" to bypass Gatekeeper

### Windows

1. Download the `PDF-Splitter-windows-amd64-v1.0.0.zip` file
2. Extract the ZIP file to a folder of your choice
3. Double-click `PDF-Splitter.exe` to run the application
4. If Windows Defender blocks the executable, click "More info" and "Run anyway"

## Notes

- The executables are built using PyInstaller and include all necessary dependencies
- No Python installation required to run the application
- Some antivirus software may flag PyInstaller executables as suspicious - this is a false positive
- For source code and building instructions, see the main [README.md](../README.md)
